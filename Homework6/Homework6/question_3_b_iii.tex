Using $\lambda = \dfrac{T'(t)}{T(t)} = \dfrac{X''(x)}{X(x)}$, \say{separate} the PDE into two ODEs
\begin{align*}
    X'' &= \lambda X \tag{1}\\
    T' &= \lambda T \tag{2}
\end{align*}


\soln*

\nl (1) $$\lambda = \frac{X''}{X} \qiffq \lambda X = X''$$

\noindent (2) $$\lambda = \frac{T'}{T} \qiffq \lambda T = T'$$
