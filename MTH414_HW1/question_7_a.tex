$y = 0$ is a solution

\begin{proof}
    $y = 0 \implies y'=0 \implies y''= 0$, hence
    \begin{align*}
        y'' + P(x)y' + Q(x)y &= 0 + 0 \cdot P(x) + 0\cdot Q(x) \tag{substitution}\\
        &= 0 + 0 + 0 \tag{properties of zero}
        \\ &= 0 \tag{additive identity on $\R$}
    \end{align*}
    $\therefore \; y=0$ is a solution.  
\end{proof}
